Quelle est la bonne manière d'intégrer du CSS dans une page web ?

La bonne pratique consiste à créer un fichier séparé appelé généralement style.css. Cela rend votre code plus facilement maintenable. Vous pouvez ainsi appliquer vos styles sur plusieurs pages

A quoi sert le css ?

Le CSS nous sert à styliser notre HTML, il gére les couleurs, le positionnement, les ombres et plein d'autres choses
